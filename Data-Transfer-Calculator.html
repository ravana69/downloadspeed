What is Data Transfer Calculator?

Data Transfer Calculator is a simple web application to calculate the time it takes to transfer data when the size of the data and the speed of the network is known. It is useful if you have large amount of data to transfer, especially incase you are moving data storage centers. It is also useful for cloud system engineer to estimate the amount of data they need to transfer to each node of a CDN.

How to use?

1. Enter your transfer speed.
2. Enter the size of the data.
3. This tool will calculate the time it would take to transfer it.
